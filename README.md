## runall
并行执行命令

## Install
```
  npm i @simon_he/runall 
```

## Usage
```
# Package.json
"scripts":{
    "run-all":"runall \"pnpm lint\"  \"pnpm i\"",
    "lint": "eslint . --ext .vue,.js,.ts,.jsx,.tsx,.md,.json --max-warnings 0 --cache",
}
```
